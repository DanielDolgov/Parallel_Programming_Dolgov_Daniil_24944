# Задание 1
Описание: заполнить масив float/double значениями синуса (один период на всю длину массива), количество элементов 10^7. Посчитать сумму и вывести в терминал. Написать cmake файлы для сборки. Сделать push на GitHub. Должна быть возможность выбора типа массива (float или double) во время сборки.

## Использование
Работа в терминале, из папки задания


> Генерация по умолчанию (FLOAT)
> ```bash
> cmake -S . -B build
> ```
- Генерация с выбором FLOAT
- ```bash
- cmake -S . -B build -DFLOAT=ON
- ```
Генерация с выбором DOUBLE
```bash
cmake -S . -B build -DDOUBLE=ON
```

Сборка
```bash
cmake --build build
```

Запуск
```bash
./build/out
```

## Выводы
Сумма float: -0.0277862

Сумма double: 4.89582e-11
